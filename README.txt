== Description ==

Your Ultimate SEO Assistant. Created for SEO Specialist who do site audits and
assist them with the process making SEO fixes more fun and easy!

= Features: =

* Bulk SEO Meta Title Update
* Bulk SEO Meta Description Update
* Bulk Canonical URL Update
* Bulk Noindex/Nofollow Update for pages and posts
* Supports Yoast SEO out of the box
* Support other SEO plugins like RankMath, SEOPress, All-in-One SEO **(PRO)**
* Crawl the site and check for issues **(PRO)**
* See opportunities to improve SEO **(PRO)**
* Support Bulk Editing of WooCommerce Products **(PRO)**

== Installation ==

1. Upload `arva-seo.php` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Enter your license key or use free license
1. Enjoy!
